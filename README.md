# API Automation Project

This repository contains a **sample API automation project** designed to test REST APIs using JavaScript. The project demonstrates automated API validation on a sample public API.

## Features
- API testing of [ReqRes](https://reqres.in/api) endpoints.
- Includes tests for **GET, POST, PUT, DELETE**, and **login API scenarios**.
- Integrated with **CI/CD pipelines** for automated test execution.
- Demonstrates **data validation and response verification**.
